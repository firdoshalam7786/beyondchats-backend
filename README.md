# BeyondChats Backend

This repository contains the backend implementation for the BeyondChats assignment.

## Features
- Article scraping (Phase 1)
- CRUD APIs for articles
- AI-based article rewriting pipeline (Phase 2)

## Tech Stack
- Laravel (APIs)
- Node.js (AI script)
- Database (MongoDB / MySQL)

## Setup (Local)
- Configure `.env`
- Run backend on port 5001
- Enable CORS

## Notes
- Backend is intended to run locally for evaluation.
- Frontend consumes these APIs.
